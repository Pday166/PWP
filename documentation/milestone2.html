Content Strategy

This site will be one page and will consist of the following sections:

Welcome

Will display a brief greeting and set the tone for the rest of the page.

About Me

This section will provide a brief introduction of myself and relevant skills and experience. It will state my name and detail my core skill set and professional highlights. Professional and skill related content will be presented graphically by use of SVG icons while text content positioned adjacent to the graphical content will give a brief description further elaborating upon key points

Portfolio

Portfolio projects will be displayed in an image gallery.

Contact

This section will feature a simple contact form. This section will also provide graphical links to GitHub, LinkedIn etc.
